void setup_dac ( void )

Initializes the DAC hardware.

This function sets up the DAC hardware, including the timer and the output pins. It also initializes the DAC latency buffer and the ring buffer for the DAC output, in case DAC_LATENCY > 1.

◆ start_dac()

void start_dac ( void )

Starts the DAC hardware.

This function enables the DAC hardware and starts the timer. It also sets up the interrupt for the DAC update.
