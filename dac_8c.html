void setup_dac ( void )

Initializes the DAC hardware.

This function sets up the DAC hardware, including the timer and the output pins. It also initializes the DAC latency buffer and the ring buffer for the DAC output, in case DAC_LATENCY > 1.

◆ start_dac()

void start_dac ( void )

Starts the DAC hardware.

This function enables the DAC hardware and starts the timer. It also sets up the interrupt for the DAC update.

◆ tim3_update_it_callback()

void tim3_update_it_callback ( void )

Interrupt handler for TIM3 update interrupt.

Handles the DAC timing update interrupt. This function is called when the TIM3 update interrupt occurs. It clears the interrupt pending bit and updates the DAC output values based on the current state of the ring buffer. If DAC_LATENCY is greater than 1, it also sets a flag to indicate that new DAC values are ready to be processed.
