Detailed Description

Hardware configuration for the project.

This file contains the hardware configuration settings for the project. It includes definitions for CPU frequency, maximum prescaler divisor, maximum counter value, and includes configuration files for different programs. The configuration files are specific to the functionality of each program, such as wavetable LFO, blinky, sample hold, and master clock.
